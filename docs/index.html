▸ addRequest(id, cmd, options)
src/DeviceManager.js

Send a request to a device and get the response. If the device is not found it will attempt a refresh.

addRequest(id: string, cmd: string, options: object): Promise<string>
Parameters
id (string) id of the device to send a request to
cmd (string) command to send to the device
options (object) request options
Name Description
options.timeout number (default 200) Timeout in ms. This is used to know when a request is considered finished. A request is considered finished when the serial port stopped receiving data for more than the given timeout.
Returns
Promise<string>: The response to the request
